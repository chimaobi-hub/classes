css selector

css selector are use to select the html element you want to style.
they are patterns used to select the element you want to stlye

type of css selector

  • universal selector (*)
  • type selector (element name)
  • class selector (.class)
  • id selector (#id)
  • attribute selector ([atttribute])
  • psuedo-class selector (:psuedo-class)
  • psuedo-element selector(::psuedo-element)

universal sectors

the universal selector (*) selector all elements.

example

* { margin:0; padding:0; }

type selector

the type selector selects all elements of a given type.

examlpe

P{ color:blue; }

class sector

the class sector selects all elements with a specific class attribute

example

.highlight { background-color:yellow }

id selector

id selector selects a single elements with a specific id attribute

attribute selector

the attribute selector selects elemnts based on an attribute or attribute value.

psuedo-class selector

psuedo-class selector selects and styles a part of an element

psuedo-element selector

the psuedo-element selector select and styles a part of an element.
